平台：野火开发板

TIM2 4通道 输入捕获 实现PWM脉宽测量

说明：
		由TIM2进行捕获，开通了4路都能进行捕获
测试：测试好使，对RC接收机信号进行测量，精度1us


接口定义：
1.PA0――TIM2CH1


2.USART1
	PA9――TX
	PA10――RX
